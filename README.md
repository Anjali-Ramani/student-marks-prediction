# Student Marks Prediction using Linear Regression

## 📌 Project Overview

This is my first Machine Learning project built using Python, NumPy, and Scikit-learn.

The model learns the relationship between the number of hours studied and the marks obtained by students using the Linear Regression algorithm.

## 🛠️ Technologies Used

- Python
- NumPy
- Scikit-learn

## 📊 Dataset

| Hours Studied | Marks |
| ------------- | ----- |
| 1             | 20    |
| 2             | 35    |
| 3             | 50    |
| 4             | 65    |
| 5             | 80    |

## 🎯 Prediction

The model predicts the marks of a student who studies **6 hours**.

**Predicted Marks:** 95.0

## 📈 Output

This project predicts student marks based on study hours using Linear Regression.

The project also visualizes the dataset using a scatter plot and displays the Best Fit Line generated by the trained model.
